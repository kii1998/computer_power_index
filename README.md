# GAC-Index: 全球 AI 算力全产业链指数

本项目通过 Python 自动抓取美股主要 AI 算力产业链公司历史数据，按自定义权重归一化计算出“全球 AI 算力全产业链指数（GAC-Index）”，并可视化为交互式 K 线图。

## 特性
- 成分股涵盖 GPU、晶圆、服务器、云、能源等全产业链龙头
- 指数归一化，基准日点位为 100，权重可自定义
- 自动计算全组合真实成交金额（Turnover）
- Plotly 高品质交互式 K 线与成交金额副图
- 支持一键导出 CSV 结果

## 快速开始

1. 安装依赖

```bash
pip install yfinance pandas plotly
```

2. 运行脚本

```bash
python gac_index.py
```

3. 结果说明
- 运行后会自动下载数据、计算指数、生成交互式图表，并导出 `gac_index_ohlc_turnover.csv`
- 图表可缩放、平移，成交金额以十亿美元为单位

## 文件结构

- `gac_index.py`      主程序，包含全部逻辑
- `GAC-Index.ipynb`   Jupyter 交互式分析版本
